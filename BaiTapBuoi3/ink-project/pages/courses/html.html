Tìm hiểu HTML:

HTML (viết tắt của HyperText Markup Language - Ngôn ngữ đánh dấu siêu văn bản) là ngôn ngữ nền tảng được sử dụng để xây dựng cấu trúc của mọi trang web trên Internet

Bạn có thể hiểu về HTML một cách đơn giản như sau:

  • Là "bộ khung" của website: Nếu ví website như một ngôi nhà, thì HTML chính là phần khung xương, cột trụ và gạch xây tường. Nó định nghĩa đâu là tiêu đề, đâu là đoạn văn, hình ảnh hay đường link.
  • Không phải ngôn ngữ lập trình: HTML là một ngôn ngữ "đánh dấu". Nó dùng các thẻ (tags) để bao quanh nội dung, giúp trình duyệt web (Chrome, Safari,...) hiểu được nội dung đó cần hiển thị như thế nào.

Vai trò trong bộ ba phát triển Web

HTML hiếm khi đứng một mình mà thường kết hợp với:

  1. HTML: Tạo cấu trúc và nội dung (Bộ khung)
  2. CSS: Định dạng và kiểu dáng (Màu sắc, font, bố cục)
  3. JavaScript: Thêm tương tác và tính năng động (Sự kiện, animation)
